## 💬 Sobre mí

Soy un apasionado desarrollador Java con formación en Desarrollo de Aplicaciones Multiplataforma (DAM). Desde que completé la parte teórica del ciclo, me he especializado en el ecosistema Spring, profundizando en Spring Boot para crear aplicaciones backend escalables, limpias y mantenibles.

📘 Mi compromiso con la calidad del código me llevó a estudiar Clean Code (Robert C. Martin), lo que transformó mi forma de pensar y estructurar soluciones.

🤝 Durante mis prácticas y el Proyecto Final, trabajé en entornos colaborativos reales, reforzando tanto mis habilidades técnicas como las blandas.

🚀 He participado en Hackathones de Java y contribuido a proyectos open source, donde he demostrado mi capacidad de autoaprendizaje y adaptación constante.

🎯 Actualmente, no hay tecnología que me intimide si está alineada con mis objetivos profesionales. Estoy en constante evolución, siempre buscando nuevos desafíos para crecer como desarrollador.
